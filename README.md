<h1 align="center">in1t</h1>
<p align="center">
  <b>A simple and fast APT manager made in shell.</b>
</p>

**in1t** is only compatible with **MacOS**, **Linux**, and **iOS (Jailbroken)**. It is **not** compatible with Windows.

# Installation

To install in1t on **MacOS and Linux**, you can clone this repository by running the following command:
```
git clone --recursive https://github.com/NotHansCYDIa/in1t && cd in1t
```
After cloning the repository, you need to download a release or file from the releases section. Alternatively, you can download it by clicking [here](https://github.com/NotHansCYDIa/in1t/releases/tag/init-v1.0.0) . After that, you can run the following command to set up **in1t**:

```sh
chmod +x setup.sh && ./setup.sh /Users/MyUser/Downloads/in1t_1.0.0_unitful
```

Make sure to replace `/Users/MyUser/Downloads/in1t_1.0.0_unitful` with the directory where you have installed the `in1t` installation file. Once you have executed the command, you should be able to run `in1t`.
