<h1 align="center">in1t</h1>

<p align="center">A fast and simple open-source APT Package manager</p>

<br>

**in1t** is a **fast and simple open-source APT package manager** made in shell. in1t is compatible on **iOS *(jailbroken)*, macOS and Linux**. This version of in1t is the advanced version of in1t.

<details><summary><b>Switch to legacy in1t</b></summary>

  If you don't want to use **in1t**, then you can switch to [legacy in1t](https://github.com/NotHansCYDIa/in1t/tree/legacy).

</details>

# Installation
To install in1t, open Command Prompt/Terminal. And run
```
git clone --recursive https://github.com/NotHansCYDIa/in1t && cd in1t
```
after you do that it should automatically go to the in1t directory. Make sure to install the in1t installation file *(in1t_installation_1.0.0, etc.)*, then run `chmod +x setup.sh` then after that run `./setup.sh in1t_installation_1.0.0`. Replace `in1t_installation_1.0.0` into the directory where you downloaded the installation file. After you press enter you shouls now be able to run `in1t`. If you want to see all commands then run `in1t -h`.
